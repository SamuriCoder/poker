# Royal Flush - Texas Hold'em Poker (Multiplayer)

A real-time multiplayer Texas Hold'em poker game. Host a game, share the code, and play with friends.

## How to Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the server:
   ```bash
   npm start
   ```

3. Open your browser to `http://localhost:3000`

4. To play multiplayer:
   - **Host**: Click "Host Game" → share the game code → when players have joined, click "Start Game"
   - **Join**: Enter the game code and click "Join"

## Tech Stack

- Node.js + Express (static files and HTTP)
- Socket.IO (real-time WebSocket communication)
- Vanilla JavaScript, HTML, CSS
